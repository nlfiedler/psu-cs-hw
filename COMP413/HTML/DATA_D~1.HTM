Glossary

  • Vertex - a node specified by its (x,y) 2D coordinates
  • Edge - a pair of vertices:
    • Directed - a ordered pairs of vertices
    • Undirected - a unordered pairs of vertices
  • Graph - a list of vertices and edges:
    • Directed - a list of vertices and directed edges
    • Undirected - a list of vertices and undirected edges
    • Methods of graph representation:
      • Adjacency Matrix - a two dimensional array used to represent a graph. For each edge (u,v), set A[u,v] = 1 if the edge exists; if no edge exists, set A[u,v] = 0.
      • Adjacency List - a one dimensional list of vertices where each vertex points to another list of adjacent vertices for each vertex
      • Edge List - a list of edge costs
      • Vertex List - a list of vertices
